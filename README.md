This is a experimental project to create small projects in it to try out new and trending tech in web development
Tech Included: Prisma , Next.js14, Reactjs, TailwindCSS, Typescript etc...

## Getting Started
First, run the development server:

```bash
npm run dev

Open [http://localhost:3000/calculator](http://localhost:3000/calculator) with your browser to see Calculator Project
Open [http://localhost:3000/todolist](http://localhost:3000/todolist) with your browser to see Task Management Project
